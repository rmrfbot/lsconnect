# LoneStar Connect or lsconnect
script for new linux installs that connects to our university wifi 
REQUIREMENTS
NETWORKMANAGER
LINUX

INSTALLATION:
this uses NetworkManager so if you get errors just:

sudo apt install network-manager

OPERATION:

once you get a terminal running (or any new linux install) you just type 

sudo apt install lsconnect

then by running:

sudo lsconnect connect
